# FIS205 - Tarea 1

Resolución de la Tarea 1 del curso Física Computacional (UTFSM).

## Contenido

- `problema1_cuantica/`: dinámica cuántica y complejidad computacional en cadena de espines
- `problema2_fft/`: transformada de Fourier y comparación entre DFT directa y FFT
- `problema3_misiles/`: simulación numérica de intercepción de misiles balísticos
- `Desarrollos_Teóricos_Tarea_1_FIS205___Diego_Yáñez.pdf`: desarrollo teórico complementario

## Ejecución

El proyecto está diseñado para ejecutarse mediante scripts de Python (`.py`), no mediante notebooks (`.ipynb`). Esto permite una ejecución más directa, reproducible y modular de cada problema.

Cada problema se ejecuta de forma independiente desde su respectiva carpeta, por ejemplo:

```bash
python problema1_cuantica/main.py
python problema2_fft/main.py
python problema3_misiles/main.py
